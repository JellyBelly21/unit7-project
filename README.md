## Getting Started

1. **Clone this repo** to your computer
2. Open the folder in your editor
3. Open `index.html` in a browser

The HTML and CSS are already built. Your job is to write the JavaScript that makes everything work.

## What You're Building

A **Profile Generator** where the user:
- Fills out a form with name, age, email, hobby, vibe, and a terms checkbox
- Clicks Submit
- The form validates the input
- If something's wrong, they see a specific error message
- If everything's good, a custom profile card appears below!

## Stuck?

1. Open the browser console (F12) - check for errors first
2. Page reloading on submit? You forgot `event.preventDefault()`
3. Number math weird? Wrap in `Number()`
4. Checkbox always wrong? Use `.checked` not `.value`
5. Card not showing? Make sure you remove the "hidden" class
6. Validation not stopping at first error? Use `else if`, not multiple `if`s
7. Go back to your Unit 7 codealong file and student handout
